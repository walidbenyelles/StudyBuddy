# StudyBuddy – Study planner landing page

Landing page pour une application fictive de planification de révisions destinée aux étudiants.  
Le projet présente l’idée d’une app simple pour organiser les séances d’étude et suivre la progression.

## Fonctionnalités

- Section hero avec présentation de l’app et appels à l’action.
- Bloc illustratif montrant un planning “Today”.
- Section “Features” avec trois fonctionnalités principales.
- Section “How it works” en trois étapes.
- Formulaire “Get notified” (démo) pour laisser un email.
- Design responsive pour mobile, tablette et desktop.

## Stack technique

- HTML5  
- CSS3 (Flexbox, Grid, responsive design)

## Objectif du projet

- Illustrer une landing page d’application (type SaaS / app étudiante).
- Montrer la capacité à structurer une page marketing claire.
- Alimenter mon portfolio de développeur web front‑end.
